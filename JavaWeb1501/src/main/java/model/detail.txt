メソッド名(引数1の型 引数1, 引数2の型 引数2,…)｛｝

public int plusTax(int product)
{ return (int) (product * TAX); }
商品の価格に税率 (TAX) を掛けて、結果を int 型として返す。
・product : 商品価格
・TAX : 税率（今回は1.1）final intで定数で定義した。
・product * TAX : 税込み価格を計算
・(int) : キャスト。小数点以下を切り捨てて整数に変換
